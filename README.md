# PmuMentor<br/>
PmuMentor: A Peer Mentorship App for PMU students PmuMentor is a peer mentorship online platform for university students where more experienced students (mentors) can offer guidance, support, and knowledge to less experienced students (mentees). This platform aims to facilitate academic success, career advice, and personal development.<br/>
<br/>
A preliminary list of features: A user having a PmuMentor account can log into the system to:<br/>
1- Create an account<br/>
2- Update account information<br/>
3- Create virtual or in-person events like workshops, webinars, etc.<br/>
4- Share study guides, articles, tutorials, and information about internships, scholarships, and career opportunities.<br/>
5- chat between the mentor and the mentee <br/>
6- basic search bar where users can search mentor name and major or event title<br/>
7- Light/Dark Mode Toggle (this can't be done without Java script and no backend is required from us thats why we won't add this in the current website) <br/>
8- Filter Events by Type<br/>
9- Access a Help/FAQ page explaining how to use the platform<br/>
10- Rate mentors and provide feedback after sessions or events<br/>
