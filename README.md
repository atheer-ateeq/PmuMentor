# PmuMentor
PmuMentor: A Peer Mentorship App for PMU students PmuMentor is a peer mentorship online platform for university students where more experienced students (mentors) can offer guidance, support, and knowledge to less experienced students (mentees). This platform aims to facilitate academic success, career advice, and personal development. 
 A preliminary list of features: A user having a PmuMentor account can log into the system to:
1- Create an account
2- Update account information
3- Create virtual or in-person events like workshops, webinars, etc.
4- Share study guides, articles, tutorials, and information about internships, scholarships, and career opportunities.
5- chat between the mentor and the mentee 
6- basic search bar where users can search mentor name and major or event title
7- Light/Dark Mode Toggle
8- Filter Events by Type
9- Access a Help/FAQ page explaining how to use the platform
10- Rate mentors and provide feedback after sessions or events
